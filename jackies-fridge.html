First-time Setup

Step 1 — Create the Google Sheet

1

Go to sheets.google.com and create a new blank spreadsheet. Name it Fridge Stock.

2

In Row 1, type these six headers exactly as shown, one per column:
id   name   category   zone   drawer   qty

3

In the menu bar go to Extensions → Apps Script. A new tab opens.

4

Delete all the default code in the editor. Open the fridge-apps-script.gs file you downloaded, copy everything, and paste it into the Apps Script editor. Click the 💾 Save icon.

5

Click Deploy → New deployment. Under Select type choose Web app. Set Execute as to Me and Who has access to Anyone. Click Deploy and authorise when asked.

6

Copy the Web app URL that appears (it starts with https://script.google.com/macros/s/…) and paste it below.
