First-time Setup

Connect to Google Sheets

1

Go to sheets.google.com — create a blank sheet named Fridge Stock. In Row 1 add these headers:
id name category zone position qty

2

Go to Extensions → Apps Script. Delete the default code, paste in the fridge-apps-script.gs contents and save.

3

Click Deploy → New deployment → Web app. Set Execute as Me and Who has access Anyone. Deploy, authorise, copy the URL.
